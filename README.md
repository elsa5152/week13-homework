* Create a .gitignore file and include node_modules/ 

* $ npm init

* $ npm install i sequelize

* $ npm i dotenv

*  MySQL2 package

*  console.table package 

*  add .env file 

*  node seeds/index.js

  Run CODE " npm start " IN COMMAND-LINE.
